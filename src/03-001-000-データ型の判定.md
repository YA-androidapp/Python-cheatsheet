## データ型の判定

type()と isinstance()がある。継承を考慮する必要があるかで使い分ける。
以下の例では、bool は int のサブクラスなので、isinstance で判定すると継承元の型にも True を返す

| 関数                      | 結果    |
| ------------------------- | ------- |
| `type(False) is bool`     | `True`  |
| `type(False) is int`      | `False` |
| `isinstance(False, bool)` | `True`  |
| `isinstance(False, int)`  | `True`  |
