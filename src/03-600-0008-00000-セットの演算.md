### セットの演算

```py
s1 = {'ab', 'cd'}
s1 = {'ef', 'ab', 'cd'}
```

| 関数      | 値                   |
| --------- | -------------------- |
| `s1`      | `{'ab', 'cd'}`       |
| `s2`      | `{'ef', 'ab', 'cd'}` |
| `s1 - s2` | `set()`              |
| `s1 | s2` | `{'ab', 'cd', 'ef'}` |
| `s1 & s2` | `{'ab', 'cd'}`       |
| `s1 ^ s2` | `{'ef'}`             |
