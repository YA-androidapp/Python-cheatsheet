#### int 型のフォーマット

フォーマットに利用する関数には以下の種類がある

- 組み込み関数`format()`
- `str.format()`
- - f 文字列

| 種別           | 書き方                                    | 結果       |
| -------------- | ----------------------------------------- | ---------- |
| `format()`     | `format(255, '06o')`                      | `'000377'` |
| `str.format()` | `'{:06o}'.format(255)`                    | `'000377'` |
| f 文字列       | `f'right : {255:06o}'`                    | `'000377'` |
|                | `strval=255`<br>`f'right : {strval:06o}'` | `'000377'` |

詳細は[書式指定子](#書式指定子)の項を参照
