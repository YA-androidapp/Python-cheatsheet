### 絶対値（abs）

| 関数         | 引数の型   | 結果の型 |
| ------------ | ---------- | -------- |
| abs(x)       | int        | int      |
|              | float)型   | float    |
|              | complex)型 | float    |
| math.fabs(x) | int        | float    |
|              | float)型   | float    |

```py
abs(-123)
abs(-1.23)
abs(32 + 1j)
```

> 123
>
> 1.23
>
> 32.01562118716424

```py
import math
math.fabs(-123)
math.fabs(-1.23)
math.fabs(32 + 1j)
```

> 123.0
>
> 1.23
>
> TypeError: can't convert complex to float
