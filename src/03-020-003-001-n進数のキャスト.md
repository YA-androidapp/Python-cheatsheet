#### n 進数のキャスト

接頭辞(0b, 0o, 0x)がついていれば、基数に 0 を指定しても変換される

| n 進数  | 関数                   | 結果    |
| ------- | ---------------------- | ------- |
| 2 進数  | `int('0b11111111', 2)` | `255`   |
|         | `int('0b11111111', 0)` | `255`   |
| 8 進数  | `int('0o777', 8)`      | `511`   |
|         | `int('0o777', 0)`      | `511`   |
| 16 進数 | `int('0xffff', 16)`    | `65535` |
|         | `int('0xffff', 0)`     | `65535` |
