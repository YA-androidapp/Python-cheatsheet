##### メタ文字

| 文字 | 内容（ `re.ASCII` を指定した場合） | 内容（Unicode (str) パターン）                                 |
| ---- | ---------------------------------- | -------------------------------------------------------------- |
| `\b` | 文字列の境界                       | 単語は `\w` からなる（英数字以外も含む）                       |
| `\B` | `\b` 以外                          |                                                                |
| `\d` | `[0-9]`                            | `[0-9]` とその他多数の数字                                     |
| `\D` | `\d`                               |                                                                |
| `\s` | `[ \t\n\r\f\v]`                    | Unicode 空白文字                                               |
| `\S` | `\s` 以外                          |                                                                |
| `\w` | `[a-zA-Z0-9_]`                     | 単語の一部になりうるほとんどの文字、数字、およびアンダースコア |
| `\W` | `\w` 以外                          |                                                                |

| 文字 | 内容             |
| ---- | ---------------- |
| `\0` | NULL 文字        |
| `\A` | 文字列の先頭     |
| `\f` | FF               |
| `\l` | `[a-z]`          |
| `\L` | `\l` 以外        |
| `\n` | LF               |
| `\r` | CR               |
| `\t` | TAB              |
| `\u` | `[A-Z]`          |
| `\U` | `\u` 以外        |
| `\v` | VTAB（垂直タブ） |
| `\Z` | 文字列の末尾     |

| 文字         | 内容                 |
| ------------ | -------------------- |
| `\N{name}`   | Unicode 文字（名前） |
| `\uxxxx`     | （16bit）            |
| `\Uxxxxxxxx` | （32bit）            |
