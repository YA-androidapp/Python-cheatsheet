#### str.format 関数

```py
print('{}'.format(1))
print('{{}}'.format(1)) # {}自体を記述したい場合は2つ重ねる
```

> 1
>
> {}

```py
print('{:+}'.format(1)) # 正の数でも符号を表示

print('{:0=10}'.format(100)) # ゼロ埋め
print('{:010}'.format(100))
print('{:0=10}'.format(-100))
print('{:010}'.format(-100))

print('{:<10}'.format(1)) # 左寄せ

print('{:>10}'.format(1)) # 右寄せ
print('{:0>10}'.format(1))

print('{:^10}'.format(1)) # センタリング(中央寄せ)
print('{:0^10}'.format(1))
print('{:*^10}'.format(1))
```

- 入れ子

```py
val = 15
digits = 6

'{:0>{width}b}'.format(val, width=digits)
'{0:0>{1}b}'.format(val, digits)
'{:0>{}{}}'.format(val, digits, 'b')
'{:{}{}{}}'.format(val, '0>',digits, 'b')
```
