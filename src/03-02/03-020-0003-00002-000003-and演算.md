##### AND 演算

| 関数                   | 結果       | 備考                                  |
| ---------------------- | ---------- | ------------------------------------- |
| `bin(bin(0b0 & 0b0))`  | `'0b0'`    |                                       |
| `bin(bin(0b0 & 0b1))`  | `'0b0'`    |                                       |
| `bin(bin(0b1 & 0b1))`  | `'0b1'`    |                                       |
| `bin(0b1100 & 0b1010)` | `'0b1000'` | 繰り上がりせず、各桁ごとに AND される |
