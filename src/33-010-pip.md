## pip

### パッケージのインストール

#### pip 自体のアップデート

```sh
$ pip install -U pip
```

#### requirements.txt を基にしたインストール

```sh
$ pip install -r requirements.txt
```

#### 特定のバージョンのインストール

```sh
$ pip install <package>==<version>
```

#### requirements.txt の書き出し

```sh
$ pip freeze > requirements.txt
```

### パッケージのアンインストール

```sh
$ pip uninstall -y <package>
```
