##### 整列

| 関数                                                         | 値      |                        |
| ------------------------------------------------------------ | ------- | ---------------------- |
| `print('{} {} {}'.format(1, 2, 3))`                          | `1 2 3` | 複数の値を埋め込む     |
| `print('{2} {0} {1}'.format(1, 2, 3))`                       | `3 1 2` | 埋め込み順序を指定する |
| `print('{one} {three} {two}'.format(one=1, two=2, three=3))` | `1 3 2` | 埋め込み順序を指定する |
