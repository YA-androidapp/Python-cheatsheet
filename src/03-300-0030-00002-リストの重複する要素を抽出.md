#### リストの重複する要素を抽出

```py
# 順番を無視
l = ['foo', 'bar', 'hoge', 'foo', 'bar', 'hoge', 'foo', 'bar', 'hoge']
sc = [x for x in set(l) if l.count(x) > 1]
print(sc)

# 順番を保存
# Python 3.6以降
df = [x for x in dict.fromkeys(l) if l.count(x) > 1]
print(df)

# Python 3.5以前
sk = sorted([x for x in set(l) if l.count(x) > 1], key=l.index)
print(ss)
```

> \# 順番を無視
>
> ['hoge', 'bar', 'foo']
>
> \# 順番を保存
>
> ['foo', 'bar', 'hoge']
>
> ['foo', 'bar', 'hoge']
