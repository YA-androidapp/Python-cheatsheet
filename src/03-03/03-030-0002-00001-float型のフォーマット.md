#### float 型のフォーマット

フォーマットに利用する関数には以下の種類がある

- 組み込み関数`format()`
- `str.format()`
- - f 文字列

| 種別           | 書き方                                        | 結果      |
| -------------- | --------------------------------------------- | --------- |
| `format()`     | `format(1.2345, '0.3f')`                      | `'1.234'` |
| `str.format()` | `'{:0.3f}'.format(1.2345)`                    | `'1.234'` |
| f 文字列       | `f'right : {1.2345:0.3f}'`                    | `'1.234'` |
|                | `strval=1.2345`<br>`f'right : {strval:0.3f}'` | `'1.234'` |

詳細は[書式指定子](#書式指定子)の項を参照
