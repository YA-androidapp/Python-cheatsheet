#### 算術演算子

```py
# 和（加算）
print(1.23 + 2)

# 差（減算）
print(1.23 - 2)

# 積（乗算）
print(1.23 * 2)
```

> 3.23
>
> -0.77
>
> 2.46

```py
# 商（実数除算）… Python2 では / でも整数除算になる
print(1.23 / 2)

# 商（整数除算）
print(1.23 // 2)

# 剰余（<ruby>余り<rt>あまり</rt></ruby>）
print(1.23 % 2)

# 商と剰余
divmod(1.23, 2)
```

> 0.61
>
> 0.0
>
> 1.23
>
> (0.0, 1.23)

```py
# べき乗（累乗）
print(1.23 ** 2)
```

> 15129
