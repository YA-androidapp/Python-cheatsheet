### 四捨五入・数値の切り上げ・切り捨て

| 処理                      | 関数                | 結果 |
| ------------------------- | ------------------- | ---- |
| 四捨五入                  | `round(7 / 3)`      | `2`  |
| 切り捨て                  | `7 // 3`            | `2`  |
|                           | `int(7 / 3)`        | `2`  |
| 切り上げ                  | `-(-7 // 3)`        | `3`  |
| 切り捨て(math モジュール) | `math.floor(7 / 3)` | `2`  |
| 切り上げ(math モジュール) | `math.ceil(7 / 3)`  | `3`  |

```py
import math
math.floor(7 / 3) # 切り捨て
math.ceil(7 / 3) # 切り上げ
```
