### 現在日時

```py
from datetime import datetime

dt = datetime.today()
print(dt)

dt = datetime.now()
print(dt)
```

> 2019-08-02 08:34:17.354115
>
> 2019-08-02 08:34:17.354115

| 項目              | 値                             | 備考           |
| ----------------- | ------------------------------ | -------------- |
| `type(dt)`        | `<class 'datetime.datetime'\>` |                |
| `dt.year`         | `2019`                         |                |
| `dt.month`        | `8`                            |                |
| `dt.day`          | `2`                            |                |
| `dt.weekday()`    | `4`                            | 0:月曜; 6:日曜 |
| `dt.isoweekday()` | `5`                            | 1:月曜; 7:日曜 |
| `dt.hour`         | `8`                            |                |
| `dt.minute`       | `34`                           |                |
| `dt.second`       | `17`                           |                |
| `dt.microsecond`  | `354115`                       |                |
