## 型の宣言

```py
#!/usr/bin/python3
# -*- coding: utf-8 -*-

boolvar: bool = True
intvar: int = 123
floatvar: float = 1.23
strvar: str = 'foo'

strvar = 'bar'
boolvar = 'hoge'
```
